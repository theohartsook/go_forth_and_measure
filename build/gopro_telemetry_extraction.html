gopro_telemetry_extraction.applyTags(input_dir, imu_csv, gps_csv, config_file, file_ending='.jpg', west_hem=True)¶

Tags a directory of cts labelled frames with their GPS and IMU data using exif_tool.

Parameters

  • input_dir – Filepath to the directory of labelled frames.

  • imu_csv (str) – Filepath to the IMU data (must be cleaned with cleanIMU())

  • gps_csv (str) – Filepath to the GPS data (must be cleaned with cleanGPS())

  • fps (int) – FPS of the original video.

  • config_file – Filepath to the config file for exif_tool.

  • file_ending (str) – Target ending for the frames, defaults to ‘.jpg’

  • west_hem (bool) – Controls whether longitude is written in W hemisphere, defaults to True

gopro_telemetry_extraction.extractAllFrames(input_video, output_dir, prefix='frame_', sig_fig=7, file_ending='.jpg')¶

Extracts all frames from a video using ffmpeg.

Parameters

  • input_video (str) – Filepath to the target video

  • output_dir (str) – Filepath to the directory where the tagged frames will be stored.

  • prefix (str) – Prefix for the extracted frames, defaults to ‘frame_’

  • sig_fig (int) – Controls the length of the frame IDs, defaults to 7 (i.e. 0000001-9999999)

  • file_ending (str) – Sets the frame output format, defaults to ‘.jpg’
